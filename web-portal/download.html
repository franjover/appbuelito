Como instalar

1

Descarga el archivo

Pulsa el boton de arriba. Se descargara un archivo APK en tu movil.

2

Permite la instalacion

Android te pedira permiso para instalar apps de fuentes externas. Pulsa "Permitir" o "Ajustes" y activa la opcion.

3

Instala y abre

Pulsa "Instalar" cuando aparezca el dialogo. Una vez instalada, abre Appbuelito y sigue el proceso de configuracion inicial.
